# 标杆主题清单

## 主题 1: 缓存对象粒度与数据结构
- 维度 1.1: 缓存对象粒度（候选：inode 级 / dentry 级 / inode+dentry 混合 / 路径级）
- 维度 1.2: 索引数据结构（候选：Hash Table / Radix Tree / B+Tree / 多级索引）
- 维度 1.3: 属性存储方式（候选：完整 inode 副本 / 增量属性 / 分层存储）

## 主题 2: 缓存一致性协议
- 维度 2.1: 一致性模型（候选：强一致性（租约） / 最终一致性（失效通知） / 弱一致性（TTL）/ 混合模型）
- 维度 2.2: 失效传播机制（候选：主动推送（callback） / 被动拉取（版本号） / 租约过期 / 混合）
- 维度 2.3: 并发写冲突处理（候选：写穿透（write-through） / 写回（write-back） / 写失效（write-invalidate））

## 主题 3: 缓存淘汰策略
- 维度 3.1: 淘汰算法（候选：LRU / LFU / ARC / LIRS / 2Q / 分层淘汰）
- 维度 3.2: 容量控制维度（候选：条目数上限 / 内存字节数上限 / 混合限制）
- 维度 3.3: TTL 策略（候选：固定 TTL / 自适应 TTL / 分类 TTL / 无 TTL）

## 主题 4: 预取与批量优化
- 维度 4.1: 预取触发策略（候选：顺序预取 / 空间局部性预取 / 基于访问模式预测 / 无预取）
- 维度 4.2: 批量查询粒度（候选：单次 RPC 批量查询 / 流式批量查询 / 目录级批量预取 / 子树批量预取）
- 维度 4.3: 预取深度控制（候选：固定深度 / 自适应深度 / 按工作负载类型调整）

## 主题 5: 性能与资源权衡
- 维度 5.1: 内存开销控制（候选：固定容量池 / 动态扩缩容 / 分级缓存）
- 维度 5.2: 缓存命中率优化目标（候选：最大化命中率 / 最小化延迟 / 平衡命中率与内存）
- 维度 5.3: 热点数据识别（候选：访问计数 / 时间衰减 / 工作集检测 / 无识别）
