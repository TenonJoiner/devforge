# 禅道 CLI 技能

通过 zentao 命令行工具查询和操作禅道（ZenTao）数据，覆盖项目集、产品、项目、执行、需求、Bug、任务、测试用例、测试单、产品计划、版本、发布、反馈、工单、应用、用户、附件等模块的增删改查及状态流转。

## 功能亮点

- 命令行操作禅道，无需打开网页
- 支持 AI Agent 直接调用（Claude Code, Cursor, VS Code 等）
- 自动处理分页、认证、数据过滤
- 支持 CRUD 及状态流转操作

## 适用场景

- 查询项目进展、Bug 列表、任务分配
- 快速创建需求、Bug、任务
- 更新对象状态（激活/关闭/解决等）
- 批量数据处理和导出
