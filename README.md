# DevForge

分布式存储开发团队技能仓库 (原 teamskills)

## 项目定位

覆盖产品规划到代码交付全流程的 skill 体系，用于分布式存储团队的 AI 辅助开发工作流。

## 目录结构

- .claude/ - Claude Code 配置
  - agents/ - 自定义 Agent
  - commands/ - 自定义命令
  - skills/ - 自定义 Skills
  - rules/ - 规则文件
- reference/ - 参考仓库（不上传，见 .gitignore）

## 使用方式

各子系统代码仓通过引用方式使用本仓库提供的统一 skill/agent/command/hook。
