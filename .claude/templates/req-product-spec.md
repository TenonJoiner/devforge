| Actor | 权力（高/低） | 利益（高/低） | 策略 | 需求关注点 |

| | | | | |

矩阵策略说明：
- **高权力+高利益**：重点满足，保持密切协作
- **高权力+低利益**：令其满意，避免反对
- **低权力+高利益**：持续告知，争取支持
- **低权力+低利益**：最小投入，监控即可
